# 人生K线图

将八字命理转化为金融K线图，直观展示百年人生运势的牛市与熊市。

## When to Use This Skill

当用户要求以下任务时使用：
- 生成个人的人生K线图 / 运势走势图
- 将八字分析结果可视化为K线
- 想要看到未来各年运势的涨跌趋势
- 想要多维度评估（事业、财运、姻缘、健康等）
- "帮我画个运势K线图"、"人生走势怎么样"等请求

## Instructions

### 第一步：收集信息

需要用户提供：
1. **公历出生日期**（年月日）
2. **出生时间**（精确到分钟最佳）
3. **性别**（影响大运方向和维度评分）
4. **姓名**（可选，用于图表标题）

如果用户同时安装了 `tianji` 技能，可以先用 tianji 完成八字排盘，再用本技能生成K线图。

### 第二步：准备输入数据

只需准备最少信息，四柱八字由脚本自动计算：

```json
{
  "name": "张三",
  "gender": "男",
  "solar_date": "1990-05-20",
  "birth_time": "08:30"
}
```

四柱自动计算算法：
- **年柱**：以立春为界（天文算法计算节气日期）
- **月柱**：以 12 个"节"为分界 + 五虎遁元推月干
- **日柱**：儒略日数算法
- **时柱**：五鼠遁元 + 早子时方案

如提供 `bazi` 字段，脚本会逐柱对比校验并提示差异。

### 第三步：运行计算

```bash
python scripts/lifeline_k.py --input data.json --output result.json --html chart.html
```

脚本会：
1. 自动计算完整四柱八字（年柱、月柱、日柱、时柱）
2. 计算大运序列（10组以上）
3. 逐年计算流年干支，生成 OHLC K线数据
4. 计算八大维度评分（1-5分）
5. 检测高能年份（巅峰/低谷/转折/爆发）
6. 生成结构化人生阶段分析
7. 输出 JSON 数据 + 可交互 HTML 图表

### 第四步：输出结果

#### JSON 数据（result.json）

包含完整数据：

**timeline** — 逐年K线：
- `year`/`age`：年份和虚岁
- `open`/`close`/`high`/`low`：K线 OHLC 值（0-100）
- `da_yun`/`liu_nian`：当前大运和流年干支
- `trend`：牛（上升）或 熊（下降）

**summary** — 结构化分析：
- `bazi_summary`：八字概要（四柱、日主、性别）
- `dim_scores`：八大维度评分（1-5分，0.5步长）
- `highlights`：高能年份列表（类型、描述）
- `life_phases`：人生阶段划分（少年/青年/壮年/中年/晚年，含均分和评级）
- `statistics`：整体统计（牛熊比、极值等）

#### HTML 图表（chart.html）

独立的可交互 HTML 文件，特点：
- 使用 ECharts CDN，无需安装任何依赖
- 支持鼠标悬停查看每年详情（年份、岁数、OHLC、大运、流年）
- 支持缩放和拖拽浏览
- 自动标记人生最高峰和最低谷
- 底部有逐年运势明细条形图
- 深色主题，金融风格
- **可直接在 Coze 的卡片/WebView 中渲染**

### 第五步：解读报告

将 HTML 图表分享给用户，同时在对话中输出结构化解读。

#### 1. 多维度评分表格

基于 result.json 中的 `summary.dim_scores`，输出评分表格：

```
| 维度 | 评分 | 说明 |
|------|------|------|
| 事业 | ★★★★☆ (4.0) | 事业发展与职业运 |
| 财运 | ★★★½ (3.5) | 财富积累与理财运 |
| 姻缘 | ★★½ (2.5) | 感情婚姻与伴侣缘 |
| 健康 | ★★ (2.0) | 身体健康与精力 |
| 学业 | ★★★ (3.0) | 学业智慧与进修运 |
| 人际 | ★★½ (2.5) | 人际关系与贵人运 |
| 子女 | ★★★½ (3.5) | 子女缘分与亲子关系 |
| 精神 | ★★½ (2.5) | 心态幸福与精神状态 |
```

评分星级渲染规则：
- 满分 = ★，半分 = ½ 或 ☆，空 = ☆
- 例：3.5 → ★★★½☆

#### 2. 高能年份解读

基于 `summary.highlights`，逐一解读：
- ⭐ 人生巅峰：最佳机遇期，适合大展拳脚
- ⚠️ 人生低谷：韬光养晦期，需防范风险
- 🔄 大运转换：人生方向转变点
- 🚀 最大涨幅：爆发年份
- ⚡ 最大跌幅：需特别注意

#### 3. 人生阶段总结

基于 `summary.life_phases`，按阶段解读运势走向：
- 少年期（1-18岁）→ 成长环境、学业基础
- 青年期（19-35岁）→ 事业起步、感情发展
- 壮年期（36-55岁）→ 事业黄金期、家庭稳固
- 中年期（56-70岁）→ 收获期、转型期
- 晚年期（71岁+）→ 晚运、子女缘

#### 4. 整体运势概要

基于 `summary.statistics`，给出：
- 百年均分和牛熊比例
- 最佳十年和最差十年
- 命理特征总结

## 八大维度评分算法

基于传统命理十神六亲对应关系，综合原局十神分布（40%）、大运十神影响（40%）和长生旺衰（20%）计算。

| 维度 | 关键十神 | 命理依据 |
|------|---------|---------|
| 事业 | 正官、偏官、正印 | 官星主事业权力，印星主学历地位 |
| 财运 | 正财、偏财、食伤 | 财星主财富，食伤生财为财源 |
| 姻缘 | 正财/正官（按性别） | 男看正财（妻），女看正官（夫） |
| 健康 | 正印、比肩（利）/官杀（忌） | 印星护身，官杀克身 |
| 学业 | 正印、偏印、食伤 | 印星主文，食伤主智慧创造力 |
| 人际 | 食神、比肩、正印 | 食神主福禄亲和，比肩主同辈 |
| 子女 | 官杀/食伤（按性别） | 男看官杀（子女），女看食伤（子女） |
| 精神 | 食神（利）/偏官（忌） | 食神主乐观，偏官主压力 |

评分 1-5 星（0.5步长），基于绝对校准，不同命盘的同一维度可横向对比。

## K线运势评分算法（v3.0）

采用"大运基准 + 流年涟漪"双层结构 + EMA 平滑，生成有明显波段的K线：

### 大运基准（慢变层）
| 因子 | 权重 | 说明 |
|------|------|------|
| 大运天干十神 | ×2.5 | 决定10年主趋势方向 |
| 大运地支长生 | ×1.8 | 帝旺/临官旺，死/绝/病弱 |

### 流年涟漪（快变层）
| 因子 | 权重 | 说明 |
|------|------|------|
| 流年天干十神 | ×0.8 | 年度小幅波动 |
| 流年地支长生 | ×0.6 | 年度地支影响 |
| 大运流年交互 | ±2 | 天干之间的十神关系 |
| 地支合冲 | ±2.5~3 | 六合加分、六冲扣分 |

### EMA 平滑
- 平滑因子 α=0.35，使价格曲线跟随大运趋势
- 大运切换年 α=0.55，加速过渡使转折更明显
- OHLC: Open=上年Close, Close=EMA平滑值, High/Low=基于偏差的波动

## Important Notes

- 本技能使用确定性算法（非 AI 随机生成），同一八字多次运行结果一致
- 四柱全部自动计算：节气用天文算法（USNO 太阳位置 + UTC+8），日柱用儒略日算法
- 采用早子时方案（23点不换日），与主流排盘网站一致
- 运势分数和多维评分仅供参考娱乐，命理学并非精确科学
- HTML 图表使用 ECharts CDN，需要网络连接才能渲染
- 脚本为纯 Python 3，无第三方依赖
