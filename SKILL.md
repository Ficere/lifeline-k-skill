# 人生K线图

将八字命理转化为金融K线图，直观展示百年人生运势的牛市与熊市。

## When to Use This Skill

当用户要求以下任务时使用：
- 生成个人的人生K线图 / 运势走势图
- 将八字分析结果可视化为K线
- 想要看到未来各年运势的涨跌趋势
- "帮我画个运势K线图"、"人生走势怎么样"等请求

## Instructions

### 第一步：收集信息

需要用户提供：
1. **公历出生日期**（年月日）
2. **出生时间**（精确到分钟最佳）
3. **性别**（影响大运方向）
4. **姓名**（可选，用于图表标题）

如果用户同时安装了 `tianji` 技能，可以先用 tianji 完成八字排盘，再用本技能生成K线图。

### 第二步：准备输入数据

需要确认以下八字信息（可手动排盘或从 tianji 结果中获取）：

- **四柱八字**：年柱、月柱、日柱、时柱
- **农历月日**：用于称骨等辅助参考

日柱由脚本内置的 `calc_day_pillar()` 函数自动校验（基于儒略日算法）。

准备输入 JSON：

```json
{
  "name": "张三",
  "gender": "男",
  "solar_date": "1990-05-20",
  "birth_time": "08:30",
  "bazi": ["庚午", "辛巳", "乙酉", "庚辰"],
  "lunar_month": 4,
  "lunar_day": 26
}
```

### 第三步：运行计算

```bash
python scripts/lifeline_k.py --input data.json --output result.json --html chart.html
```

脚本会：
1. 自动校验日柱正确性
2. 计算大运序列（10组以上）
3. 逐年计算流年干支
4. 综合十神关系、长生十二诀、地支六合六冲推算运势分数
5. 生成 OHLC（开盘-收盘-最高-最低）数据
6. 输出 JSON 数据 + 可交互 HTML 图表

### 第四步：输出结果

#### JSON 数据（result.json）

包含完整时间线，每年有：
- `year`/`age`：年份和虚岁
- `open`/`close`/`high`/`low`：K线 OHLC 值（0-100）
- `da_yun`/`liu_nian`：当前大运和流年干支
- `trend`：牛（上升）或 熊（下降）

#### HTML 图表（chart.html）

独立的可交互 HTML 文件，特点：
- 使用 ECharts CDN，无需安装任何依赖
- 支持鼠标悬停查看每年详情（年份、岁数、OHLC、大运、流年）
- 支持缩放和拖拽浏览
- 自动标记人生最高峰和最低谷
- 底部有逐年运势明细条形图
- 深色主题，金融风格
- **可直接在 Coze 的卡片/WebView 中渲染**

### 第五步：解读报告

将 HTML 图表分享给用户，并提供解读：

1. **整体趋势**：指出人生的主要上升和下降阶段
2. **关键转折点**：大运交替带来的运势转换
3. **牛市阶段**：绿色K线集中的时期，解释为何运势走强
4. **熊市阶段**：红色K线集中的时期，提醒注意事项
5. **峰值年份**：标星的最高点，解释其命理原因

## 算法说明

运势评分综合以下6个维度：

| 维度 | 权重 | 说明 |
|------|------|------|
| 大运天干十神 | ×1.5 | 正财/正官/正印加分，偏官/劫财扣分 |
| 大运地支长生 | ×1.2 | 帝旺/临官旺，死/绝/病弱 |
| 流年天干十神 | ×2.0 | 每年变化的主要波动来源 |
| 流年地支长生 | ×1.5 | 每年运势的地支影响 |
| 大运流年交互 | ±3 | 天干之间的十神关系 |
| 地支合冲 | ±4~5 | 六合加分、六冲扣分 |

OHLC 生成规则：
- Open = 上一年 Close（连续性）
- Close = 当年综合得分
- High/Low = 基于波动率计算的上下影线

## Important Notes

- 本技能使用确定性算法（非 AI 随机生成），同一八字多次运行结果一致
- 日柱由内置的儒略日算法自动计算，无需第三方服务
- 运势分数仅供参考娱乐，命理学并非精确科学
- HTML 图表使用 ECharts CDN，需要网络连接才能渲染
- 脚本为纯 Python 3，无第三方依赖
